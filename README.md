# A-Rhythmic-Imprint

**DATA AS MATERIAL**
A rain-sensing umbrella that records 10-second rainfall sessions and stores them as generative musical memories.
Rainfall data determining the musical scale the umbrella produces

## Features
- Session-based rainfall recording
- Archive of up to 5 sessions
- Generative playback using scale mapping
- Serial Plotter visualization mode

## Controls
- r = record new session
- n = next session
- a1, a2, a3... = choose session
- p = toggle plot mode
