# A-Rhythmic-Imprint

An umbrella that records rainfall impact and treats data as material, translating environmental intensity into generative sound and rhythmic memory.


## Features
- Session-based rainfall recording
- Archive of up to 5 sessions
- Generative playback using scale mapping
- Serial Plotter visualization mode

## Controls
- r = record new session
- n = next session
- a1, a2, a3... = choose session
- p = toggle plot mode
